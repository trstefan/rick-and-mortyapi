## :runner: Run Locally

Clone the project

```bash
  git clone https://github.com/trstefan/rick-and-mortyapi
```

Go to the project directory

```bash
  cd rick-and-mortyapi
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
